# Java Calculator Application

A simple console-based calculator built using Java. This application allows users to perform basic arithmetic operations through an interactive menu-driven interface.

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Modulus Operation
* Error Handling for Division by Zero
* Continuous User Interaction

## Technologies Used

* Java
* Scanner Class
* Switch Case
* Loops
* Exception Handling

## How to Run

1. Compile the program:
 
javac Calculator.java

3. Run the program:

java Calculator
